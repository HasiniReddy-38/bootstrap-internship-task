About This Project

This internship task was completed by exploring Bootstrap 5 official examples and creatively remixing components into a beautiful single-page application.

I studied the Album, Pricing, and Blog examples to extract patterns for navbar, hero, cards, modal, and form — then combined them with custom styling for visual synergy.

  • Smooth scrolling navigation
  • Working contact form (Formspree)
  • Hover animations & active navbar
